Install Jenkins

Install Java

Jenkins is a Java web application which need at least JRE 1.6

Check your java version

                        
                            java -version

Install Git

Git is the new leading source control software, use it!

                        
                            sudo apt-get install git

for Windows user install git from here

Check git install

                        
                            git --version

Forking the Sample Repository

                        
                            git clone git@github.com:username/game-of-life.git

or 

                        
                            https://github.com/wakaleo/game-of-life.git

Dowload and install Jenkins

  • Dowload the jenkins.war
  • Launch jenkins 
                                        
                                        java -jar jenkins.war
  • See what happen if we go to your favorite browser (ie not IE... ) 
                                        
                                        http://localhost:8080/jenkins
